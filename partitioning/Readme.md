##### English:
Scripts in this folder are designed to partition tables in **Oracle Database**, which contains [Zabbix](http://www.zabbix.com) DB.
Tested on:

* Oracle 11gR2
* Zabbix Server 2.0.X

Проверено на:

* Oracle 11gR2
* Zabbix Server 2.0.X
